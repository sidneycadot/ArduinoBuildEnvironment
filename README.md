
GOALS
=====

(1) A working cross-compiler GCC build environment for the ARM-based Arduino Due.
(2) A working newlib, uclib, or other small C library\
(3) The appropriate CMSIS stuff for the ARM chip
(4) A working FreeRTOS
(5) Flash tool
